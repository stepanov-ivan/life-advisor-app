## Signals that help the application

- public repository

## Strong positioning for this repository

Life Advisor is not just another chat wrapper around an LLM. The core open-source value is the combination of:

- explicit WHO-aligned nutrition rules
- explainable rule evaluation
- local-first user data storage
- LLM-assisted structured meal parsing
- a testable hybrid architecture that others can study and reuse

## Good use cases to mention in the application

- generating and maintaining regression tests for nutrition rules
- reviewing parsing edge cases and malformed LLM outputs
- helping triage user-reported meal-estimation failures
- supporting release checklists, issue triage, and documentation updates

- domain importance
