## ADDED Requirements

### Requirement: Глобальная кнопка входа в агента
Система SHALL показывать одну глобальную CTA-кнопку агента по центру нижней части основного приложения как основной вход в agent-first сценарии.

#### Scenario: Кнопка доступна из любой вкладки
- **WHEN** пользователь находится на любой основной вкладке приложения
- **THEN** в нижней центральной позиции отображается круглая CTA-кнопка агента

#### Scenario: Один активный agent flow
- **WHEN** в приложении уже активна agent-сессия
- **THEN** система не открывает вторую независимую agent-сессию поверх первой

### Requirement: V1 открывает meal chat напрямую
Система SHALL в первой версии открывать по CTA-кнопке сразу meal-oriented chat input без промежуточного выбора сценария.

#### Scenario: Прямой вход в meal chat
- **WHEN** пользователь нажимает глобальную CTA-кнопку агента
- **THEN** открывается chat input для meal-сценария как основной entry point

#### Scenario: Planning остаётся внутри chat input
- **WHEN** пользователь отправил сообщение в meal chat и система строит план
- **THEN** пользователь остаётся в chat input состоянии до получения результата planner-а
