## ADDED Requirements

### Requirement: Локальные подсказки ввода
Система SHALL показывать подсказки ввода только из последних 50 структурированных логов пользователя без онлайн-запроса к LLM на каждый ввод.

#### Scenario: Показ подсказок по истории
- **WHEN** пользователь начинает вводить текст приёма пищи
- **THEN** система предлагает варианты из последних 50 структурированных логов

### Requirement: Ранжирование подсказок
Система SHALL ранжировать подсказки по комбинации частоты использования и свежести.

#### Scenario: Ранжированный список
- **WHEN** найдено несколько подходящих подсказок
- **THEN** выше отображаются более частые и более свежие варианты

### Requirement: Ограничение числа подсказок
Система SHALL показывать не более 5 подсказок в выпадающем списке.

#### Scenario: Лимит подсказок
- **WHEN** найдено больше 5 совпадений
- **THEN** отображаются только первые 5 подсказок

### Requirement: Подстановка структуры из подсказки
Система SHALL при выборе подсказки подставлять текст и связанную структуру, но не отправлять лог автоматически.

#### Scenario: Выбор подсказки
- **WHEN** пользователь нажимает на подсказку
- **THEN** текст и структура заполняются, отправка выполняется только после явного нажатия кнопки сохранения

#### Scenario: Визуальная маркировка источника
- **WHEN** структура подставлена из подсказки
- **THEN** в UI отображается явный бейдж `Из памяти`

### Requirement: Сброс структуры при очистке текста
Система SHALL сбрасывать связанную структуру, если пользователь полностью очистил текст после подстановки подсказки.

#### Scenario: Очистка текста
- **WHEN** поле ввода очищено до пустого состояния
- **THEN** связанная структура удаляется из черновика

### Requirement: Canonical-шаблоны и alias
Система SHALL хранить один canonical-шаблон структуры и список alias-текстов для эквивалентных формулировок.

#### Scenario: Две формулировки одного блюда
- **WHEN** пользователь использует разные текстовые формулировки с одной и той же структурой
- **THEN** система связывает их с одним canonical-шаблоном и обновляет список alias
