## MODIFIED Requirements

### Requirement: Объяснимая память питания
Система SHALL хранить минимальную объяснимую память питания, включающую исходную фразу приёма пищи и нормализованные структурные позиции, пригодные для прямого заполнения канонического `MealEvent.items`.

#### Scenario: Сохранение сигнала питания
- **WHEN** пользователь сохраняет структурированный приём пищи
- **THEN** система сохраняет структурный сигнал в памяти с привязкой ко времени и источнику структуры

#### Scenario: Сохранение фразы и структуры вместе
- **WHEN** формируется memory-запись приёма пищи
- **THEN** система сохраняет и исходную фразу, и нормализованные позиции в одной записи памяти

#### Scenario: Использование памяти как источника структуры
- **WHEN** пользователь выбирает memory-подсказку
- **THEN** система использует нормализованные позиции памяти для заполнения структуры без агрегированного «слитого» формата

#### Scenario: Невалидная memory-запись
- **WHEN** memory-запись не содержит валидную исходную фразу
- **THEN** система исключает запись из подсказок без fallback и пишет debug-лог

#### Scenario: Фильтр подсказок памяти
- **WHEN** система формирует список memory-подсказок
- **THEN** в список попадают только записи с валидными `sourceText`, `items` и normalized-текстом длиной не менее 3 символов

#### Scenario: Нет валидных memory-подсказок
- **WHEN** после фильтрации список memory-подсказок пуст
- **THEN** система показывает только свободный ввод без отдельной заглушки блока памяти

### Requirement: Контроль памяти пользователем
Система SHALL предоставлять точечное удаление элемента памяти и полный сброс памяти.

#### Scenario: Точечное удаление
- **WHEN** пользователь удаляет конкретную запись памяти
- **THEN** запись удаляется и перестаёт использоваться в подсказках и рекомендациях

#### Scenario: Полный сброс
- **WHEN** пользователь запускает полный сброс памяти
- **THEN** все memory-записи удаляются

#### Scenario: Destructive reset change
- **WHEN** выполняется destructive reset под новый формат
- **THEN** память и история приёмов пищи очищаются полностью
